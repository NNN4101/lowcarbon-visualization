一个基于数据驱动的交互式分析平台，旨在揭示中国各省份在能源结构、生态绿化与碳排放效率之间的协同演化关系。

系统特点：
三大分析模块：空间分布识别、协同机制分析、时序演化追踪；
多源数据整合：整合CEADs碳排放、国家统计局能源、住建部绿化等多维度数据；
交互式可视化：采用Vue3 + ECharts构建丰富的图表联动与数据下钻体验；
全栈技术实现：前端Vue + 后端Flask的完整数据产品解决方案。

技术栈
前端：Vue3 + TypeScript + ECharts + TailwindCSS；
后端：Python Flask + Pandas + Scikit-learn；
数据：2005-2022年中国省级面板数据；
